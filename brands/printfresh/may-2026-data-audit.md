## Reviews (Yotpo, 404 reviews)

**Score distribution:** 5★ 351 (87%), 4★ 23 (6%), 3★ 13 (3%), 2★ 10 (2.5%), 1★ 7 (1.7%). Average ~4.78.

**What customers love (5-star themes):**
"Soft," "comfortable," "cozy," "beautiful prints," "luxurious," "obsessed." Specific silhouettes praised: Wildest Dreams, Rittenhouse, Sweet Dreams, Soho.

**Critical findings. Flag to client (not test ideas):**

1. **"New sizing" change is causing returns and complaints.** Multiple reviews call out a sizing reset vs. prior orders:
   - "Not a fan of your new sizing"
   - "Wildest Dreams... first set I ordered in the new sizing. It is slightly big and next time I will go back to the M"
   - "Fits smaller than previous pjs - sizing was much different than before. I got my usual size large"
   - Multiple "too big," "huge," "runs really big" mentions across silhouettes

2. **Quality regression among long-time customers.**
   - "What is going on with the fabric quality? I've ordered over a dozen pairs of pajamas over the years"
   - "Not the usual printfresh quality and felt scratchy"
   - "Fabric felt rough and was sheer- not at all what I have come to expect"

3. **Eco Satin description misleads customer expectations.**
   - "It's not the shiny satin feel or appearance you may think of. It is a viscose and a muted color. Wrinkles after you wash it and shouldn't be described as satin"

4. **See-through items surprise on final sale.**
   - "Completely see through - I wish this was made explicit in the description - especially for final exchange"

5. **Final sale + sizing mismatch creates angry customers.**
   - "Disappointed. I checked the size chart... it's non-exchangeable final sale"
   - "Unable to exchange due to sizing being out of stock"

**Product mix demand signals (client-actionable):**
- Print available in more silhouettes ("Hope this print is produced in a future Eco Satin or long PJ version")
- Cropped pants in eco satin (not just cotton)
- Full sheet sets (currently only pillowcases)
- Pink Bagheera comforter set return
- Kids sizes for Must Love Cats

**Wins to amplify:**
- New **Pima Modal Knit (Luna Jersey)** launch is landing well: "Printfresh has hit a home run with this new Pima fabric"
- **Tall sizing wins fans when available**: "I can't thank print fresh enough for doing the tall sizing!"
- **Gifting use cases dominate** mentions: weddings (bridesmaid getting-ready, honeymoon), birthdays, mothers, hostess gifts. Validates bride-bridesmaid collection (37K sessions/mo).

## Slot Selection Logic

Three slots, AOV must include at least one. Each slot must be distinct (different page or different element).

| Slot | Page | Themes Targeted | Data Strength | Why this over alternatives |
|---|---|---|---|---|
| 1: HSS Activation Components | Site-wide bar + HSS collection pages | Theme 1, Theme 3 | 5 sources | Client #1 priority, time-bounded. Component-level activations reuse across 4 sale events without per-sale rebuild cost. Avoids LP build that can't bake in 5 days. |
| 2: Post-purchase upsells (Intelligems) | Order confirmation flow | Theme 2 | 4 sources + tooling | AOV anchor (mandatory). Cart upsells already tested at Printfresh and lost; post-purchase is a net-new surface via Intelligems' new feature. Zero pre-purchase conversion risk. |
| 3: PDP enrichment stack | PDP (top-traffic products) | Themes 4, 5 | 5+5 sources | 480K direct PDP entries. Fabric story + Shop the Print + colorway swatches. V1 leads with fabric story (not reviews-above-fold, since brand-level rating is already clickable on PDP). |

**Tests considered but not selected for this month:**
- HSS landing page (full LP build): not selected. Build cost vs. 5-day window doesn't justify, and the LP can't accumulate significance in time. Component-level activations capture the same demand and reuse across all 4 sale events.
- Cart drawer upsell module: not selected. Already tested at Printfresh and lost. Post-purchase upsells via Intelligems take the AOV slot instead.
- Athos-based PDP upsells: dependent on Athos setup, expected June 2026. Not in May scope.
- Reviews-above-fold on PDP: not selected. Brand-level star rating already renders clickable on PDP, so this is not a productive variation.
- Hen House LP test specifically: strong volume but compounds with HSS activation framework (similar pattern, similar data). Logical next-month follow-up if HSS framework wins.
- Filter / nav UX overhaul: multiple verbatims but scope is bigger than one slot, requires dev cycles. Future slot candidate.
- Gift workflow rebuild: "Gifting Made Easy" promise vs. real gaps. Possibly a multi-slot dev project for Q3.
- Sizing confidence module on PDP: already tested without significant lift per client; not the right swing this month.
- Homepage shoppable tiles: no heatmap data yet on click distribution or per-tile RPS, so the test variable is unclear. Request heatmap access before scoping.
- Mobile gallery zoom: user-flagged UX issue but needs a structured walkthrough to articulate the specific friction before turning into a test.
