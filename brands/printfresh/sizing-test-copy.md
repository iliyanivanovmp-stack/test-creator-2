# Sizing Test Copy by Fabric (Test #5: PDP Review-Powered Sizing Confidence)

Reference for the fabric-specific fit note in the "How It Fits" strip below the size selector (V6 mockup).

---

## Eco Satin
> Eco Satin runs relaxed. Size down if between sizes.

**Status:** Already in V6 mockup. Confirmed by reviews and site copy.

## Cotton (Organic Cotton Poplin)
> Cotton fits true to size. Go with your usual size.

**Fit notes:** Most TTS of all Printfresh fabrics. Multiple product pages and third-party reviews confirm.

## Flannel
> Flannel fits true to size. Go with your usual size.

**Fit notes:** Generally TTS. One reviewer noted flannel fits slightly more generous than cotton poplin, but overall consensus is true to size. Petite flannel may run a touch smaller than regular.

**Alternative copy if we want to differentiate from cotton:**
> Flannel fits true to size. If between sizes, either will work.
